How this index is supposed to work

Treat this page as a neutral launchpad. Each workspace can evolve independently—the URLs here are the only coupling. When you add a new HTML file, you only need to:

  • Drop the new *.html file alongside this index or in its folder.
  • Add a button here pointing at that file (like the ones on the left).
  • Keep names stable so links never break.

Common flows this index supports:

Start with multi-cloud decision logic Jump into Terraform maps Pivot to Ansible & day-2 ops Log decisions in SBRs / ATOS

You can reorder the buttons or rename labels without breaking anything—only the href="...html" targets need to match the actual filenames.
